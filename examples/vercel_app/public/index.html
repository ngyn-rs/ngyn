Welcome to Ngyn

Ngyn is a progressive backend framework written in Rust. It's designed to be fast, reliable, and efficient, making it ideal for high-performance web applications.

Installation

To install Ngyn, you need to have Rust installed on your machine. Once you have Rust installed, you can install Ngyn using cargo: 

cargo install ngyn

Capabilities

Ngyn can handle routing, middleware, and async interactions. It's designed to be extensible, allowing you to add your own functionality as needed.

Limitations

While Ngyn is powerful, it's not a silver bullet. It's not designed to handle frontend logic or user interface design. For those tasks, you'll need to use a frontend framework or library.
